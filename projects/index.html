Hi I'm Ben and I'm a website developer. Some of my projects are listed below.

NCAA Site Predictor

The NCAA basketball tournament releases the locations of games less than one week before the tournament begins. Using the NCAA Net Rankings this program predicts where top teams will be placed, and has an interactive feature for fans to see different possible scenarios.

Technologies Used

HTML, CSS, JavaScript, jQuery, nodeJS

NCAA Scoreboard

This scrapes the NCAA website for live basketball scores without the heavy footprint that other score projects have. It's great for checking scores while in a low data area.

Technologies Used

HTML, CSS, JavaScript, jQuery, nodeJS

Coat Check

This is a web app that lets the user keep track of a coat check. It uses local storage to persist the data, and the user can download the data into a .csv file. The data is sortable by clicking on table headers.

Technologies Used

HTML5, CSS, JavaScript, jQuery

Vecta Corp

Vecta Corp is a fictional company used for projects in the Front End Web Developer Certificate program through the San Diego Community College District. This is an example of a responsive website home page.

Technologies Used

Skeleton framework, CSS media queries
